# Subscription Tiers & Entitlements

## 1. Trial Tier
**Target Audience:** Users exploring the platform.
**Goal:** Guided exploration, easy upgrade path.

**Features & Limitations:**
- **Access:** All tabs/modules accessible for exploration.
- **Data/Processing:** Very low limits; cannot create final deliverables (models, 360 tours, etc.).
- **SlateDrop Storage:**
  - Automated Folders: One for each tab (Project Hub, Design Studio, etc.), plus "General" and "History".
  - *Constraint:* Automated folders cannot be edited/moved/deleted.
  - User Folders: Users can create subfolders with full permissions (rename/copy/move/delete/share).
- **Upgrade Path:** Data and projects must carry over seamlessly upon upgrade.

## 2. Creator Tier
**Target Audience:** Content creators for themselves or clients.
**Focus:** Visual content (Content Studio, 360 Tour Builder).

**Features:**
- **Access:** Content Studio, 360 Tour Builder.
- **Capabilities:**
  - Create and share projects/tours via permission-based folder system.
  - Send 360 tour links to clients.
  - Embed support: Clients can embed tours on their own websites (hosted by Slate360).
  - Data Usage Widget: Track consumption.
  - Credit System: Fast credit purchase (saved card).
- **SlateDrop Storage:**
  - Automated Folders: Content Studio, 360 Tour Builder, General, History.
  - *Constraint:* Automated folders locked.
  - User Folders: Full control.

## 3. Model Tier
**Target Audience:** Advanced creators, modelers, drone operators.
**Focus:** 3D processing, geospatial, parametric design.

**Features:**
- **Access:** Design Studio, Content Studio, 360 Tour Builder, Geospatial & Robotics.
- **Capabilities:**
  - **Server-side Processing:** Photogrammetry, geolocation data, aligned LiDAR point clouds.
  - **Design:** Parametric tools (SketchUp-like).
  - **3D Print Lab:** Print management to single/multiple printers.
  - **Animation:** 3D model progressions (start to finish).
  - **Robotics:** Drone flight programming, volume calculations.
- **SlateDrop Storage:**
  - Automated Folders: Design Studio, Content Studio, 360 Tour Builder, Geospatial & Robotics, General, History.
  - *Constraint:* Automated folders locked.
  - User Folders: Full control.

## 4. Business Tier
**Target Audience:** Contractors, Project Managers.
**Focus:** Construction management, documentation, reporting.

**Features:**
- **Access:** All tabs.
- **Project Hub:**
  - Setup: Location (Google Sat), Time/Phases, Project Team (Stakeholders), Scope/Budget/Schedule.
  - Execution: Administer/Track/Document progress.
  - Documents: RFI/Submittals flow.
  - Visuals: Tours/Models for inspection.
- **Deliverables:**
  - Analytics flow for auto-populated reports.
  - Bid package creation.
- **SlateDrop Storage:**
  - Automated Folders: All tabs.
  - **Projects Folder:** Specialized structure where each Project Hub project gets a dedicated subfolder hierarchy.
  - Context Awareness: Opening a file should link deep into the tool/screen where usage is intended.
  - *Constraint:* Automated folders locked.

## 5. Enterprise Tier
**Target Audience:** Large organizations, multi-team firms.
**Focus:** Seat management, collaboration, advanced controls.

**Features:**
- **Access:** All tabs (configurable per seat).
- **Capabilities:**
  - **Seat Management:** Assign full/restricted licenses to employees.
  - **Collaboration:** Real-time multi-user editing (Design Studio), plan reviews.
  - **Admin:** Manage team access and permissions across locations.
- **SlateDrop Storage:** Same as Business Tier (Projects folder structure + All tabs).

## Competitive Benchmarks (Must Match/Exceed)
- **Project Hub:** Procore
- **Design Studio:** SketchUp, Bluebeam, Revit, Autodesk (Upload/Model/Plan Review)
- **Content Studio:** CapCut, Final Cut Pro
- **360 Tour Builder:** Kuula
- **Geospatial & Robotics:** DroneDeploy, Pix4D, Propeller

## Accessibility & Localization
- **Global Sales:** Language and Unit selections (Metric/Imperial) accessible from Homepage/Account Creation.
- **UI/UX:** Simple, intuitive, few clicks.
- **Help:** Hoverable icons, FAQs, User Guide.
- **AI Transparency:** Disclaimers on all AI results ("AI is not perfect, please verify").
