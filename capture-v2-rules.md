# SITE_WALK_CAPTURE_V2_V0_RULES

1. v0 must not create a new global app shell.
2. v0 must not create a normal mobile bottom nav.
3. v0 must not create a dashboard.
4. v0 must design capture interface content only.
5. capture-v2 must use Graphite Glass style.
6. capture-v2 must support Quick Walk and Plan Walk.
7. capture-v2 must include:
   - Action Hub
   - camera/photo/file entry
   - photo review
   - markup tools
   - details/classification with photo reference visible
   - horizontal stop filmstrip
   - multi-angle support
   - Save & Continue
   - Save & Return to Plan
   - optional Ghost/Before-After entry point
8. Old broken UI must not be imported or reused for visual layout.
9. Reuse working hooks/logic only.
